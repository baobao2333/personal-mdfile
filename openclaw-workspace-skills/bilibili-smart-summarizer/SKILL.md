  B站视频智能摘要技能。自动完成B站视频的全流程内容提取：优先使用官方字幕和B站AI总结，

## 适用场景
- 任何需要获取B站视频内容总结的场景
- 无字幕视频内容提取
- 热门视频快速内容梳理

## 执行流程
1. **元数据拉取**：调用 `bilibili video` 获取视频基础信息（标题、BV号、播放数据、UP主信息），同时尝试获取字幕和B站官方AI总结
2. **智能回退**：若字幕和AI总结均为空，自动触发 `bilibili content` 音频转写，获取完整音频文本内容
3. **补充信息**：自动拉取视频TOP10高赞热评作为观众视角补充
4. **摘要合成**：将所有获取到的文本内容整合，生成统一的结构化视频摘要

## 参数要求
| 参数名 | 是否必填 | 说明 |
| :--- | :--- | :--- |
| `target` | 是 | 视频BV号或完整B站视频URL |

## 输出格式
```
### 视频标题：<标题>
### BV号：<BV号>
### 播放数据：<播放> / <点赞> / <硬币> / <收藏>
### 内容摘要：<结构化摘要文本>
### 观众热评：<TOP3高赞热评>
```

name: tencent-ui-pro
description: >
  腾讯风格专属UI生成技能。基于腾讯QQ Design设计规范生成前端界面代码，
  内置完整的腾讯深色/浅色主题色板、字体层级、圆角、阴影规范。
  当用户要求生成腾讯风格UI、QQ风格界面、腾讯设计规范的前端页面时使用。
---

## 核心能力
1. 自动生成符合腾讯设计规范的前端界面代码
2. 内置完整的腾讯深色主题色板、字体层级、圆角、阴影规范
3. 自动校验界面审美，不符合规范的内容强制修正
4. 支持React/Vue/原生HTML多框架输出

## 使用方法
直接在prompt中描述需求，自动输出符合腾讯设计规范的前端代码：
例："生成一个直播间列表页面，使用Tencent UI Pro规范"
